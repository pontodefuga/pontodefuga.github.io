## Configurar o username
1. Abra o Git Bash (terminal)
2. Configure o Git username:
{% highlight ruby %}
$ git config --global user.name "Mr Robot"
{% endhighlight %}
3. Confirme que você configurou o Git username corretamente:
{% highlight ruby %}
$ git config --global user.name
> Mr Robot
{% endhighlight %}
